# Harvest-Kissan

A communication hub for farmers, traders, and consumers to coordinate, negotiate, and stay informed. Created in 24 hours for a hackathon (Quasar 2024)

## About

Harvest-Kissan is a lightweight project focused on connecting agricultural stakeholders to facilitate better communication, price discovery, and coordination across the supply chain. The repository contains JavaScript-based code and was created to provide a foundation for further development of a marketplace/communication platform for agriculture.

## Features

- Centralized communication channel for farmers, traders, and consumers
- Lightweight architecture suitable for rapid prototyping
- Focus on coordination, negotiation, and information sharing
